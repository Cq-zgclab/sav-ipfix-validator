# SAV IPFIX Hackathon Web 界面访问指南

## ✅ 服务已启动

HTTP服务器正在端口 8000 运行。

## 📱 访问方法

### 方法1：VS Code 端口转发（推荐）

1. 点击VS Code底部面板的 **【端口】** 标签
2. 找到端口 **8000** 这一行
3. 点击右侧的 **【在浏览器中打开】** 按钮（地球图标🌐）
4. 或者右键点击，选择"在浏览器中打开"

### 方法2：复制转发的URL

1. 在【端口】标签中找到端口 8000
2. 右键点击，选择"复制本地地址"
3. 在外部浏览器中粘贴访问

### 方法3：本地访问（仅限本机）

如果你在本地运行VS Code（非Codespace），直接访问：
- http://localhost:8000

## 🔍 测试页面

- 主界面：http://localhost:8000/index.html
- 测试页面：http://localhost:8000/test.html
- 数据文件：http://localhost:8000/data.json

## ❓ 故障排查

如果看不到页面，请检查：

1. 端口8000是否已转发
2. 浏览器是否阻止了弹出窗口
3. VS Code的Simple Browser可能有兼容性问题，建议用外部浏览器

---

**提示**：Codespaces会自动转发端口，你应该能在【端口】标签中看到一个公网可访问的URL。
